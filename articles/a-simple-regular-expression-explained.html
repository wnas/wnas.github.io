// the expression^([1-9]\d)+,?\d{0,2}// start of the line^(    // a numerical character 1 to 9 ( no 0)    [1-9]    // any numerical char    \d)// the stuff between brackets above this can be repeated. It should bethere one time or more (another possibility would be *, that wouldrequire zero times or more repeating of the above)+// an optional comma,?// some numbers can be none up to two\d{0,2}
